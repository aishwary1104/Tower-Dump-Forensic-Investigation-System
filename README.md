# Tower Dump Forensic Investigation System

## Overview
A digital forensics platform designed to analyze CDR and tower dump datasets for criminal investigations. The system uses geospatial intelligence and behavioral analytics to identify communication patterns and suspicious activities.

## Features
- CDR Analysis
- Tower Dump Processing
- Geospatial Visualization
- Behavioral Analytics
- Investigation Support Dashboard

## Technologies Used
- Python
- Pandas
- Flask
- Data Analytics

## Installation
1. Clone the repository
2. Install dependencies
3. Run the application

## Usage
Upload datasets and perform forensic analysis through the dashboard.
